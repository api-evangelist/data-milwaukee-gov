# Milwaukee Open Data (data-milwaukee-gov)

Milwaukee Open Data is a municipal government open-data portal for United States running **CKAN**, exposing a standardized catalog API over ~186 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-milwaukee-gov/refs/heads/main/apis.yml)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.11

## API
- **Milwaukee Open Data CKAN Action API** — base `https://data.milwaukee.gov/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.milwaukee.gov)
- Live endpoint verified: `https://data.milwaukee.gov/api/3/action/status_show`
